`nav2` 示例位于 `pixi` 仓库中。

```shell
git clone https://github.com/prefix-dev/pixi.git
```

进入示例文件夹

```shell
cd pixi/examples/ros2-nav2
```

运行 `start` 命令

```shell
pixi run start
```
